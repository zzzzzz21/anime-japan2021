# anime-japan2021
アニメJAPAN2021ソース管理。

# 環境構築
1. Powershellを立ち上げ。
2. ``` npm i``` を実行して開発環境をインストール
3. ```npm run start``` を実行すると、devディレクトリを生成。ローカルサーバーが立ち上がる。※以後、ファイルの更新を監視。
3. 開発作業中に画像を追加する際は、```src/images``` 下に画像を起き、```npm run imagemin``` を実行する。
4. 作業が終わったら、```npm run build``` を実行すると、納品用のdistディレクトリが作成される。
